Свойства CSS

  • color: Задает цвет текста.
  • font-size: Задает размер шрифта.
  • margin: Задает внешний отступ элемента.
  • padding: Задает внутренний отступ элемента.
  • border: Задает стиль, толщину и цвет границы элемента.
  • background-color: Задает цвет фона элемента.
  • text-align: Задает выравнивание текста.
  • display: Задает тип отображения элемента.
  • float: Управляет плаванием элемента.
  • position: Задает способ позиционирования элемента.
  • flexbox: Предоставляет гибкий способ управления распределением элементов в контейнере.
  • grid: Позволяет создавать двумерные сетки для более сложного размещения элементов.
