Селекторы CSS

Селекторы CSS - это шаблоны, используемые для выбора и стилизации элементов HTML. Вот краткий обзор:

  • *: Универсальный селектор - Выбирает все элементы на странице.
  • div: Типовой селектор - Выбирает все элементы <div>.
  • .highlight: Селектор класса - Выбирает элементы с классом highlight.
  • #important: Селектор ID - Выбирает элемент с ID important.
  • [data-custom="example"]: Селектор атрибута - Выбирает элементы с атрибутом data-custom="example".
  • article p: Селектор потомка - Выбирает все <p> внутри элементов <article>.
  • nav > ul: Селектор ребенка - Выбирает прямые потомки <ul> внутри элементов <nav>.
  • h2 + p: Селектор соседа - Выбирает <p>, следующий непосредственно после <h2>.
  • a:hover: Псевдо-класс - Стилизует ссылки при наведении на них курсора.
  • li:nth-child(odd): Псевдо-класс - Стилизует нечетные элементы списка.

Эти селекторы могут быть объединены и использованы для применения стилей к конкретным элементам или группам элементов на веб-странице.
